🤝 Рука об руку: Цифровой наставник
Цифровая платформа для обучения взрослых и пожилых людей компьютерной грамотности.
Этот проект — веб-приложение, которое соединяет молодых наставников (волонтёров) с взрослыми учениками. Платформа предоставляет интерактивные уроки, систему проверки знаний, чат с наставником и возможность оставлять отзывы.
🚀 Возможности проекта
Интерактивные уроки: 6 подробных курсов (от смартфона до Госуслуг) с теорией, примерами и практикой.
✅ Тестирование: Встроенные тесты после каждого урока с мгновенной проверкой и объяснением ошибок.
💬 Чат с наставником: Возможность задать вопрос и получить помощь в реальном времени.
📊 Прогресс: Отслеживание пройденных уроков и статистика обучения.
👥 Роли пользователей:
Ученик: Проходит уроки, пишет запросы о помощи.
Наставник: Принимает запросы, отвечает в чате, смотрит статистику.
Администратор: Управляет отзывами и пользователями.
⭐ Отзывы: Система оценок и отзывов о работе наставников.
Технологии
Backend: Python 3, Flask
Database: SQLite (через Flask-SQLAlchemy)
Frontend: HTML5, CSS3, JavaScript (Vanilla)
Security: Хеширование паролей (werkzeug.security)
