# 🚀 Цифровая платформа «Наставник»

Интерактивная образовательная платформа по цифровой грамотности и кибербезопасности для старшего поколения. Проект реализован на фреймворке **Flask** с поддержкой полноценной базы данных, интерактивных онлайн-тестов и системы личных чатов с наставниками.

---

## 🌟 Ключевые возможности

- **📁 Динамическая система уроков:** Тексты 6 оригинальных уроков вынесены в изолированный файл конфигурации `lessons.json`, что позволяет легко редактировать теорию без изменения программного кода.
- **🧠 Умные интерактивные тесты:** После каждого урока пользователь проходит закрепляющий тест. При выборе любого варианта ответа система выдает мгновенный детальный разбор ошибок или подтверждение правильности решения без перезагрузки страницы (на базе AJAX/Fetch API).
- **💬 Чат с наставниками:** Студенты могут обращаться за помощью к реальным наставникам (Алине и Александре), закрепленным за профильными направлениями.
- **📊 Учет прогресса:** Результаты успешного прохождения уроков автоматически записываются в базу данных для каждого авторизованного студента.
- **⭐ Система отзывов:** Возможность оставлять оценки и текстовые отзывы о работе платформы и наставников.

---

## 🛠️ Технологический стек

- **Backend:** Python 3.x, Flask
- **Database:** SQLAlchemy (SQLite3)
- **Frontend:** HTML5, CSS3 (Кастомные стили `main.css`/`mentor.css`), JavaScript (Async Fetch API), Bootstrap 5

## 🚀 Быстрый запуск

### 1. Клонирование и подготовка
Убедитесь, что у вас установлены **Python 3** и менеджер пакетов **pip**. Установите все необходимые зависимости одной командой:

```bash
pip install Flask Flask-SQLAlchemy Werkzeug
```

### 2. Инициализация базы данных
Для того чтобы в базе данных (`site.db`) автоматически создались все необходимые рабочие таблицы, а также появились преднастроенные профили наставников (**Алина** и **Александра**), запустите инициализирующий скрипт:

```bash
python create_mentor.py
```

### 3. Запуск веб-сервера
Запустите главное приложение платформы:

```bash
python app.py
```

🌐 Готово! Откройте ваш любимый браузер и перейдите по адресу: http://127.0.0.1:5000/

## 🔒 Безопасность и архитектура API

Архитектура платформы спроектирована с жестким упором на конфиденциальность и защиту персональных данных пользователей:

*   **Криптографическая защита данных**  
    *Компонент:* `werkzeug.security`  
    Данные учетных записей студентов и наставников никогда не хранятся в открытом виде. В базу данных (`site.db`) записываются исключительно стойкие односторонние криптографические хэши PBKDF2.
